Quick way to find the state of any instances:

aws ec2 describe-instances --output text --query 'Reservations[].Instances[].State.Name'

Quick way to grab the instance ID (assuming there's only one - otherwise you'll get more than one):

aws ec2 describe-instances --output text --query 'Reservations[].Instances[].InstanceId' > id-instance.txt

These two have been replaced with Python scripts:

( sleep 3 && aws ec2 start-instances --instance-ids $(<id-instance.txt) & ) & ./test-simple-console.py $(<id-instance.txt)
( sleep 10 && aws ec2 stop-instances --instance-ids $(<id-instance.txt) & ) & ./test-simple-console.py $(<id-instance.txt)
